# YouTube Music DL

A lightweight, powerful, and precision-engineered Chrome Extension to download music natively and directly from YouTube Music.

This extension prioritizes pristine compatibility and metadata perfection. Specifically designed to solve the long list of traditional ID3 and MP4 tagging bugs generated by web-downloaders, it ensures your grabbed tracks blend flawlessly into meticulous offline libraries like **Apple Music** and **iTunes**, effortlessly preserving all rich media tags.

## ✨ Features

*   **Native & Lightweight Capture**: Downloads and merges the exact audio stream right in your browser without requiring hefty external server rendering, `ffmpeg`, or third-party executable tools.
*   **Comprehensive Metadata Extraction**: Connects robustly to YouTube Music's DOM and MediaSession APIs to automatically extract the **Title, Artist, Album, Year, and Track Number**.
*   **Embed High-Quality Cover Art**: Features hardened image retrieval algorithms capable of circumventing CDN CORS constraints, automatically converting WebP thumbnails to strict iTunes-compliant JPEGs on the fly via the HTML5 Canvas.
*   **Complete Apple Ecosystem (macOS/iOS) Compatibility**: 
    * Automatically intercepts and blocks incompatible `WebM` streams, enforcing pure `m4a` (AAC) format captures.
    * Patched an aggressive UTF-8 byte-offset bug impacting Apple metadata atom `hdlr` and `ilst` bounds.
    * **Advanced fMP4 Brand Masking**: Incorporates an advanced byte-level Fragmented MP4 (DASH) masking patch that changes the file identity on the fly. This instantly overrides Apple's strict streaming blockers, successfully forcing Apple Music to read the local metadata block on catalog files!

## 🚀 Installation & Setup

1. Download or clone this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Toggle the **Developer mode** switch in the top right corner.
4. Click on the **Load unpacked** button and select the directory containing this repository.
5. The extension is now successfully installed! Make sure to pin it to your browser toolbar for quick access.

## 🎧 Usage

1. Open a new tab and head to `music.youtube.com`.
2. Play any song (this works for both public catalog tracks, playlists, and personal uploaded library tracks).
3. While the song is playing, click on the **YouTube Music DL** extension icon in your toolbar.
4. A popup window will appear previewing the track metadata and the downloaded cover art.
5. Choose your desired filename, or simply leave the default pattern.
6. Click **Descargar** (Download).
7. Drag and drop the downloaded `.m4a` file straight into your preferred music player—all your metadata and artwork will instantly populate!

## 🛠️ Technical Architecture

Under the hood, this extension breaks free of simple ID3 wrappers. It builds perfectly compliant ISO Base Media File Format (ISOBMFF) `moov > udta > meta > ilst` atom trees completely via Vanilla JavaScript arrays (`Uint8Arrays` and `DataViews`). In addition to parsing MP4 atoms flawlessly on-the-fly, it masks `mvex`-enabled DASH tracks with standardized `M4A ` brands allowing QuickTime systems to inherently trust and parse the files reliably without a massive re-mux operation.
